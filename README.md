# Gary Gray | Software & Game Development Portfolio

Technical portfolio demonstrating proficiency in low-level JavaScript Canvas manipulation, custom game engine architecture, and responsive UI design. This project serves as a live environment for testing real-time physics, state management, and NPC behavior.

## Deployment
Live Site: http://garysgameapps.orgfree.com/

## Technical Specifications

### Custom Game Engines (HTML5 Canvas)
* **PuckMan Engine:** Tile-based collision detection, sprite rendering, and NPC pathfinding logic. 
* **Avoider Engine:** Parallax background rendering, ship physics, and symmetrical asset generation.
* **Performance:** Built without external libraries to maintain high-frequency logic updates and low-latency rendering.

### Front-End Architecture
* **UI/UX:** Responsive grid layout using CSS Flexbox and Grid. Industrial-themed aesthetics including chromatic glitch effects and typewriter-style typography.
* **State Management:** Dynamic content injection via centralized JavaScript data objects (`textContent.js`).
* **Integrated Systems:** Implementation of custom draggable frameworks and CSS-driven environmental effects (Rain/Scroll).

### Backend Integration
* **Flask Adventure:** Demonstrates Python/Flask integration for session-driven data, user accounts, and logical consistency.

## Project Structure
* `index.html`: Core structure and dynamic script injection.
* `css/style.css`: Industrial styling, custom animations, and layout constraints.
* `js/textContent.js`: Centralized data store for portfolio content.
* `gary-bender_small.png`: Custom profile asset.

## Installation and Local Development
To view the project locally:
1. Clone the repository.
2. Navigate to the root directory.
3. Open `index.html` in a standard web browser.

Note: Certain external links and tracking assets may require a local server environment for full functionality.
